# 🏰 Royal Gruha Pravesh Digital Invitation

A premium, cinematic digital invitation built with **React**, **Vite**, and **Framer Motion**. This landing page features a royal Indian temple aesthetic with dynamic floating lanterns and scroll-triggered storytelling animations.

## ✨ Features

- **Cinematic Hero**: Optimized Cloudinary background image featuring a royal palace at sunset.
- **Floating Lanterns**: A dynamic particle system that generates glowing lanterns drifting upwards with realistic swaying motions.
- **Royal Typography**: Integration of `Cinzel` and `Lora` fonts with a custom **Gold Shimmer** animation for titles.
- **Fully Responsive**: Optimized for all devices, from small mobile viewports (320px) to ultra-wide monitors.
- **Interactive Details**: 
  - Smooth scroll-reveal animations for all sections.
  - Live **Google Maps** integration for ceremony location.
  - Glassmorphic card design for Auspicious Rituals.
  - Dedicated sections for Hosts (Invited By) and Festive Lunch.

## 🛠️ Tech Stack

- **Framework**: [React](https://reactjs.org/) (v18)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) (v3)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)

## 🚀 Getting Started

### 1. Installation
Clone the repository and install dependencies:
```bash
npm install
```

### 2. Development
Run the development server:
```bash
npm run dev
```

### 3. Build
Create a production-ready build:
```bash
npm run build
```

## 📂 Project Structure

- `src/App.jsx`: Main application logic and layout components.
- `src/index.css`: Global design tokens, Tailwind directives, and custom animations.
- `public/`: Static assets including images and textures.
